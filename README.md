## Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:Etativel/memory-card.git
   ```
2. Navigate to the project directory:
   ```bash
   cd memory-card-game
   ```
3. Install dependencies:
   ```bash
    npm install
   ```
4. Start the development server:
   ```bash
    npm run dev
   ```
5. Open the game in your browser:
   ```bash
   http://localhost:6173
   ```
